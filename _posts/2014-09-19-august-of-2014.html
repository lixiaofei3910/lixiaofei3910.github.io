工作

安装git后，用命令设置登录用户、邮箱： git config --global username "yourname" git config --global user.email "youremail" 创建ssh： ssh-keygen -t rsa -C "youremail" 将公钥添加到github上； 验证是否成功：ssh -T git@github.com

把新项目提交到github：git status git add 。 git commit -m “new” git push origin master 查看当前链接哪个仓库：git remote -v

将项目克隆到本地：git clone git@github.com:lixiaofei3910/lixiaofei3910.github.io.git

取的远程更新，合并到本地：git fetch origin

把更新的内容合并到本地：git merge origin/master
